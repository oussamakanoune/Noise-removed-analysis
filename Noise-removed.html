4. Mathematical Model from exercise 2 — updates live with N

$$u_n(x) = \frac{e^{-\sqrt{n}\,x^2}}{\sqrt{n^3}+1}, \quad x \in \mathbb{R}_+ \qquad F(x) = \sum_{n=0}^{+\infty} u_n(x)$$
$$S_N(x) = \sum_{n=0}^{N} u_n(x) \xrightarrow{N\to\infty} F(x)$$
Mapping justification (x ∈ ℝ₊):
Each frequency bin k ∈ {0, …, k_max} is mapped to x = k / k_max ∈ [0, 1] ⊂ ℝ₊
→ x = 0 (DC, 0 Hz): u_n(0) = 1/(√n³+1) → small for large n — keeps signal
→ x = 1 (cutoff): u_n(1) = e^(−√n)/(√n³+1) → decays fast — attenuates noise
→ x > 1 (above cutoff): u_n(x) → 0 exponentially — removes high-freq noise
Note on u₀(x):
u₀(x) = e^(−√0 · x²) / (√0³+1) = 1 for all x ∈ ℝ₊ (constant term).
It acts as the DC baseline: w(k) = S_N(x)/S_N(0) normalizes all weights.

10. Theory — Series of Functions Analysis 3 · Exercises 2 & 5

Pointwise convergence — weakest mode

$\forall x \in \mathbb{R}_+,\quad S_N(x) \xrightarrow{N\to\infty} F(x)$. For each fixed frequency bin (fixed x), the weighted sum converges.

Uniform convergence — preserves continuity

$\displaystyle\sup_{x \in [0,1]}\bigl|S_N(x)-F(x)\bigr| \xrightarrow{N\to\infty} 0$. All frequency bins converge at the same rate.

Normal convergence (Weierstrass M-test) — strongest & always holds here

$\displaystyle\|u_n\|_\infty = \frac{1}{\sqrt{n^3}+1}$ (attained at $x=0$).

$\displaystyle\sum_{n=1}^{\infty}\|u_n\|_\infty \leq \sum_{n=1}^{\infty}\frac{1}{n^{3/2}} < +\infty$ (Riemann series $p=3/2 > 1$).

By the Weierstrass M-test: the series converges normally on $\mathbb{R}_+$.
Normal $\Rightarrow$ Uniform $\Rightarrow$ Pointwise.

$$\|u_n\|_\infty = \frac{1}{\sqrt{n^3}+1} \leq \frac{1}{n^{3/2}} \qquad \Rightarrow \qquad \sum_{n=1}^{\infty}\|u_n\|_\infty \leq \zeta\!\left(\tfrac{3}{2}\right) \approx 2.612 \quad (\text{converges})$$
